# MDXForge Docs

MDXForge is a local-first MDX and Markdown preview tool for AI-generated documents.

It focuses on one workflow:

1. AI writes structured `.mdx` or `.md` files.
2. You open them locally in MDXForge Desktop.
3. MDXForge renders a safe, document-site-like preview.
4. You review, navigate, validate, and share the result.

## What to use it for

- Product specs
- Technical design docs
- Research summaries
- Release notes
- AI-generated documentation packs

## What not to use it for

MDXForge is not a general Markdown editor, note-taking app, hosted docs platform, or cloud collaboration product.

## Start here

Open a local `.mdx` file in the desktop app, then use the writing guide to structure richer documents with whitelisted components.
