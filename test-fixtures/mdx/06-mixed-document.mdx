<Callout title="Goal" type="info">
Use MDX as the default authoring format for Docuforge documents.
</Callout>

## Why MDX

- Easier to read than HTML
- More expressive than plain Markdown
- More controllable than arbitrary React
- Good fit for AI-generated documentation

## Workflow

```txt
AI prompt -> MDX file -> Docuforge renderer -> local preview
```
