# Callouts

<Callout title="Information" type="info">
Use this for neutral context and helpful notes.
</Callout>

<Callout title="Warning" type="warn">
Use this for risks, constraints, and important caveats.
</Callout>

<Callout title="Error" type="error">
Use this for failures, destructive actions, or invalid states.
</Callout>

<Callout type="info">
A callout can also omit the title.
</Callout>
