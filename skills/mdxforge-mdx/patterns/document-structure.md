# Document Structure Pattern

Use this structure for MDXForge docs:

1. Frontmatter with `title` and optional `description`.
2. One `#` heading matching the title.
3. Short intro paragraph.
4. Sections with `##` / `###` headings.
5. Component usage only when it improves reading or navigation.

Prefer Markdown over JSX for prose, lists, links, images, tables, and code blocks.
