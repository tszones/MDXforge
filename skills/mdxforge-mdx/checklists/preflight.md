# MDXForge Preflight Checklist

Before returning MDX:

- [ ] Frontmatter exists and includes `title`.
- [ ] No `import` or `export` statements.
- [ ] No custom React component definitions.
- [ ] No arbitrary browser/runtime expressions.
- [ ] Every JSX tag is in the MDXForge component whitelist.
- [ ] Code fences include a language.
- [ ] Unknown components were replaced with Markdown.
